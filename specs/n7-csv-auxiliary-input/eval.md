# N7: CSV 辅助输入 — 验收标准

## 验收步骤

### 1. 文件接收
- [ ] 飞书 Bot 能接收 CSV 文件附件
- [ ] 飞书 Bot 能接收 Excel (.xlsx) 文件附件
- [ ] 不支持的文件格式返回友好提示

### 2. 文件解析
- [ ] CSV 正确解析（含中文表头、不同编码）
- [ ] Excel 正确解析
- [ ] 文件 > 10MB 时拒绝并提示
- [ ] 行数 > 5000 时拒绝并提示

### 3. Context 注入
- [ ] 行数 <= 500：全量数据注入 context
- [ ] 行数 > 500：只注入列名 + 前 5 行样本
- [ ] Agent 能正确理解 CSV 内容

### 4. 结合数据库分析
```
用户上传 SKU 清单 CSV + 消息："帮我查这些 SKU 的库存"
```
- [ ] Agent 读取 CSV 中的 SKU 列表
- [ ] Agent 查询数据库获取对应库存数据
- [ ] 回复包含 CSV 中 SKU 的具体库存信息

### 5. 会话隔离
- [ ] CSV 数据只在当前会话有效
- [ ] 新建会话后 CSV context 清除

### 6. 单元测试
```bash
uv run pytest tests/test_file_handler.py -v
```
- [ ] 测试通过
