# N14: 会话超时归档 — 验收标准

## 验收步骤

### 1. 超时自动新建会话
- [ ] 用户发消息，距上一条 > 30 分钟 → 自动创建新会话
- [ ] 新会话上下文为空（不带旧历史）
- [ ] 旧会话不删除，状态变为非 active

### 2. 未超时继续会话
- [ ] 用户发消息，距上一条 < 30 分钟 → 继续当前会话
- [ ] 上下文正常保留

### 3. 配置
- [ ] session_timeout_minutes=30 → 30 分钟超时
- [ ] session_timeout_minutes=0 → 不超时（禁用功能）
- [ ] config.example.yaml 包含配置项和注释

### 4. /new 保持不变
- [ ] /new 仍可手动创建新会话
- [ ] 两种方式互不干扰

### 5. 边界情况
- [ ] 新会话（无历史消息）→ 不超时
- [ ] 首次用户 → 正常创建会话

```bash
uv run pytest tests/test_session.py -v
```
